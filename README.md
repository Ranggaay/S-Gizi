# 🥗 S-Gizi

S-Gizi adalah aplikasi mobile berbasis Flutter yang dirancang untuk membantu orang tua memantau status gizi dan pertumbuhan balita secara mudah, cepat, dan akurat menggunakan metode WHO Z-Score. Aplikasi ini dilengkapi dengan fitur monitoring pertumbuhan, rekomendasi makanan sehat, serta konsultasi dengan ahli gizi untuk mendukung tumbuh kembang balita secara optimal.

---

# ✨ Fitur Utama

## 👨‍👩‍👧 Untuk Orang Tua

* Login & Registrasi Akun
* Monitoring Status Gizi Balita
* Perhitungan Z-Score WHO
* Riwayat Pemeriksaan Gizi
* Rekomendasi Makanan Sehat
* Artikel Edukasi Gizi
* Konsultasi dengan Ahli Gizi
* Edit Profil Pengguna

## 🩺 Untuk Ahli Gizi

* Melihat Data Konsultasi
* Membalas Chat Konsultasi
* Monitoring Data Balita

## 🛠️ Untuk Admin

* Dashboard Monitoring
* Manajemen Data Balita
* Manajemen Data Makanan
* Manajemen Data Ahli Gizi
* Manajemen Artikel Edukasi
* Monitoring Hasil Pemeriksaan

# 📊 Metode Perhitungan

Aplikasi menggunakan metode WHO Z-Score berdasarkan standar pertumbuhan anak dari World Health Organization (WHO) untuk menentukan status gizi balita seperti:

* Gizi Buruk
* Gizi Kurang
* Gizi Baik
* Risiko Gizi Lebih
* Gizi Lebih
* Obesitas
